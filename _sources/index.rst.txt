   Github <https://github.com/WisconsinAutonomous/wa_simulator>

wa_simulator is a powerful, multi-platform, and *user-friendly* simulation platform for autonomous vehicle control stack development. 
wa_simulator has been built by and for the `Wisconsin Autonomous <https://www.wisconsinautonomous.org/>`_ student organization at the University of Wisconsin - Madison.

Installing
----------
wa_simulator can be installed in multiple ways. Please see our :doc:`installation guides <install>`.
