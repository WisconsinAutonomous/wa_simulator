Bases: abc.ABC

Used to manage simulation parameters and can be used to update simulation dynamics

A system is used for organizational purposes almost exclusively. It is passed to other module classes to handle passing various attributes. Could be responsible for updating module dynamics depending on the underlying models.

Parameters

  • step_size (double) – the step size for the simulation

  • render_step_size (double, optional) – Render step size. Defaults to 2e-2.
